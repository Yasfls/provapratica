# provapratica

Aluna: Yasmin Friedemann

funcionalidades:

- Cadastro de usuário com nome, email e senha
- Mensagem de boas-vindas personalizada com o nome do usuário
- Exibição dos últimos 5 livros lidos
- Navegação entre telas utilizando Navigator
- Botão de logoff, retornando à tela de login

telas:

- Tela de cadastro onde pode ser criado um usuário
- Tela de login para acessar o sistema
- Tela principal para visualizar a lista de livros


como rodar o projeto:

- Clonar o repositório
git clone https://github.com/seu-usuario/nome-do-repositorio.git

- Acessar a pasta do projeto
cd nome-do-repositorio

- Instalar as dependências
flutter pub get

- Executar o aplicativo
flutter run
